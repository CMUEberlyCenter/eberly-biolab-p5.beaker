Hash defining a conjugate base's relationship with a proton.

Type:

Properties:
Name Type Description
particle object

The proton particle, if any, joined to the conjugate base particle.
offset_x number

X position for the proton from the center of the conjugate base particle.
offset_y number

Y position for the proton from the center of the conjugate base particle.
restore_depth number

Original depth/drawing order for proton. Will be restored after separation.
release_after_range Array.<number>

An array of two numbers, used as max & min of a random duration in milliseconds for a proton to be joined.
post_release_duration number

The amount of time in milliseconds a previously joined proton & conjugate base will be unable to rejoin with other particles after separation.
release_after number

A time in milliseconds, after which a proton should be released. A non-null release_after indicates a joined or recently separated (undergoing post_release_duration) pair of conjugate base and proton.
